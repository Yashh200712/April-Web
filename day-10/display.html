Display

Display: None

Here display none means nothing to display. It will not take up any space on the page. It is as if the element does not exist in the document.

"hello display none"

Display: Block

here display block means the element will take up the full width available and start on a new line. It will create a block-level element that can contain other elements.

"hello display block"

Display: Inline

"hello display inline"

here display inline means the element will only take up as much width as necessary and will not

Display: Inline-block

here display inline-block means the element will be displayed as an inline-level element, but it will also allow you to set width and height properties. It will take up only as much width as necessary, but it can also have a specified width and height.

"hello display inline-block"
